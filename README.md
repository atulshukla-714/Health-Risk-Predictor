# Health-Risk-Predictor
A Machine Learning project that predicts health risk levels (Low, Medium, High) based on lifestyle and medical parameters such as age, BMI, smoking habits, alcohol consumption, exercise, blood pressure, and sugar levels. Built using Python, Scikit-learn, and Streamlit.

## Features Used
- Age
- BMI
- Smoking Habit
- Alcohol Consumption
- Exercise Frequency
- Blood Pressure
- Sugar Level

## Model
Logistic Regression

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Train model:
python model.py

3. Run app:
streamlit run app.py

## Output
Predicts health risk category.
