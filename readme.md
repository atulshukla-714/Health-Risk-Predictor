# \# Health Risk Predictor



\## Problem

Many people ignore early health signs. This project predicts health risk using ML.



\## Features

\- Age

\- BMI

\- Smoking

\- Alcohol

\- Exercise

\- BP

\- Sugar



\## Model Used

Logistic Regression



\## How to Run

1\. Install requirements

2\. Run model.ipynb



\## Output

Predicts: Low / Medium / High Risk
